I’m looking forward to seeing you at the upcoming Introduction to Automated Image Analysis workshop. The content covered is adapted from an Introduction to Image Analysis workshop by the Royal Microscopy Society (Barry et al., 2025). 
If you plan to attend in person, you’re welcome to bring your laptop to follow along with the exercises. To do so, please make sure to install Fiji beforehand using the instructions below. All workshop materials will be available in this GitHub repository. It may be helpful to download the contents in advance, so you’ll have everything ready to go when we begin. 

Download Instructions
1.	Download Fiji (ImageJ) HERE
To avoid any permissions issues, install FIJI is in your home directory:
-	PC: C:\users\<your username>
-	Mac: /Users/<your username>
WARNING: FIJI must be installed in a location where it has write permission - otherwise, it cannot update itself
2.	Start FIJI and allow the updater to run
3.	(Optional) If the updater does not run automatically, select Help > Update
